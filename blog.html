How Do You Select an Element from the Dom?

To select an element from the DOM, use getElementById() for elements with unique IDs or querySelector() for more flexible selection based on CSS selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach an event listener to a parent element instead of directly to each child element. This improves performance and simplifies code, especially when dealing with dynamic content.

How do you manipulate an element's attribute and styles using DOM?

To manipulate an element's attributes and styles, use getAttribute() and setAttribute() for attributes, and use the style property for inline styles.
